== Description ==

Display message or HTML content via splash popup box 

Live Preview: http://kentothemes.com/demo/kento-splash-screen/kento-splash-screen/

Plugin Features

* Unlimited Background Color.
* Unlimited Background Image.
* Custom Border Size.
* Unlimited Border Color.
* Custom HTML Content.
* Shortcode System to add any page or post.



== Installation ==

1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. After actvating you will see a menu on left side and go to "Kento Splash Screen" setting page and change your setting and save it.
4. Paste this shortcode [kento-splash-screen] in which page you want to dissplay popup box.
5. Enjoy!!!
